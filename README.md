# Indian-Tonic-Dataset
The dataset includes recordings of Indian Film music songs and few samples of classical music played on Santoor in .wav format. The corresponding tonic (Shadja) of song is provided as a label in .csv file.
Citation
Full dataset available at:https://doi.org/10.5281/zenodo.19986940
/Audio folder contains all recordings numbered from 1 to 150 in .wav file and Labels.csv file gives labels of 150 files as Tonic/scale/shadja of that song.
